# Page Transitions Example
Techonologies used:
* [GreenSock Animations](https://greensock.com/gsap/)
* [Highway Animation Manager](https://github.com/Dogstudio/highway)
* [ParcelJS](https://parceljs.org/)

## Set it up
`npm install`

## Start it up
`npm run dev`
